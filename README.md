# Blog Support Repository

This repository contains supporting resources for the **Polarpoint-io** blog articles. It is structured to host scripts and Kubernetes manifests that illustrate and accompany our blog content.

## Structure

- **/scripts/**: Contains scripts referenced in the blog articles, such as setup scripts, demos, or automation snippets.
- **/manifests/**: Includes Kubernetes YAML configurations and Helm charts tailored for deployment examples discussed in the articles.

## Purpose

The goal is to provide readers with practical and directly applicable resources to complement the theoretical insights shared in our articles.
